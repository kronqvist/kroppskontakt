# Kroppskontakt

Daily habit checklist for body-aware living. Track six focus areas each day — eating, movement, rest — and watch your consistency build over a 30-day graph.

Zero dependencies. Single HTML file. Data lives in localStorage.

## Usage

Open `index.html` in a browser. Check off items as you complete them. Navigate between days with the arrow buttons or left/right arrow keys. The graph shows your completion rate over the last 30 days.
